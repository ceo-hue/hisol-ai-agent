**실제 작동하는 검증:**

   ```typescript

     return await api.get('/data');  // ← try-catch 없음!

   ```

   // 결과: "Potential null/undefined access detected"
   ```

3. **Code Complexity**

                (if|for|while|switch 개수)

   // 15 이상이면: "High complexity - refactor needed"
   ```

4. **Code Duplication**
   ```typescript

   ```
