## 1단계. Node.js 설치

ARHA는 Node.js 위에서 실행됩니다. 먼저 Node.js를 설치해야 합니다.

### 설치 방법

1. 브라우저에서 **https://nodejs.org** 접속
2. **LTS 버전** (왼쪽 녹색 버튼) 다운로드
3. 다운로드된 `.msi` 파일 실행 → 계속 "Next" 클릭하여 설치

### 설치 확인

Windows 검색창에 `cmd` 입력 → **명령 프롬프트** 실행 후 아래 입력:

```
node --version
```

결과 예시: `v22.0.0` (18.0.0 이상이면 OK)

```
npm --version
```

결과 예시: `10.0.0` (숫자가 나오면 OK)

## 2단계. Claude Code 설치

Claude Code는 이 AI 채팅 도구입니다. 이미 사용 중이라면 이 단계를 건너뜁니다.

### 설치 방법

명령 프롬프트(cmd)에서 입력:

```
npm install -g @anthropic-ai/claude-code
```

> ⚠️ 설치 중 "권한 오류"가 나면 명령 프롬프트를 **관리자 권한**으로 실행해야 합니다.  
> (cmd 아이콘 우클릭 → "관리자 권한으로 실행")

### 설치 확인

```
claude --version
```

숫자 버전이 출력되면 성공입니다.

## 4단계. API 키 설정

ARHA는 Anthropic의 Claude AI API를 사용합니다.  
API 키가 없으면 동작하지 않습니다.

### API 키 발급

1. **https://console.anthropic.com** 접속
2. 회원가입 또는 로그인
3. 왼쪽 메뉴 **"API Keys"** 클릭
4. **"Create Key"** 버튼 클릭
5. 생성된 키 복사 (`sk-ant-api03-...` 형태)

> ⚠️ API 키는 한 번만 보여집니다. 반드시 복사해서 저장해두세요!

### API 키 등록 방법 (Windows 환경변수)

**방법 A: 영구 설정 (권장)**

1. Windows 검색창에 **"환경 변수"** 입력
2. **"시스템 환경 변수 편집"** 클릭
3. 아래쪽 **"환경 변수(N)..."** 버튼 클릭
4. "사용자 변수" 영역에서 **"새로 만들기(N)..."** 클릭
5. 변수 이름: `ANTHROPIC_API_KEY`
6. 변수 값: 복사한 API 키 붙여넣기
7. 확인 → 확인 → 확인

> 설정 후 **명령 프롬프트를 새로 열어야** 적용됩니다.

**방법 B: .env 파일로 설정**

프로젝트 폴더에 `.env` 파일 생성 후 아래 내용 입력:

```
ANTHROPIC_API_KEY=sk-ant-api03-여기에API키붙여넣기
```

파일 위치: `C:\Users\garsi\OneDrive\바탕 화면\hisol-unified-mcp\.env`

## 5단계. Claude Code에 MCP 등록

Claude Code가 ARHA 서버를 인식할 수 있도록 연결합니다.

### 등록 명령어

**새 명령 프롬프트**를 열고 아래를 한 줄씩 입력합니다.

```
claude mcp add arha-runtime node "C:\Users\garsi\OneDrive\바탕 화면\hisol-unified-mcp\dist\server.js"
```

> 성공하면 `MCP server "arha-runtime" added` 메시지가 출력됩니다.

### 등록 확인

```
claude mcp list
```

`arha-runtime` 이름이 목록에 보이면 성공입니다.

## ❗ 자주 발생하는 문제

### Q. `npm: command not found` 오류
→ Node.js 설치가 안 된 것입니다. 1단계부터 다시 진행하세요.

### Q. `Cannot find module` 오류
→ 빌드가 안 된 것입니다. 3단계에서 `npm run build`를 다시 실행하세요.

### Q. `ANTHROPIC_API_KEY is not set` 오류
→ 4단계 API 키 설정이 안 된 것입니다. 환경변수 설정 후 cmd를 새로 열어보세요.

### Q. MCP 등록했는데 이솔아가 응답 안 함
→ Claude Code를 완전히 종료 후 다시 실행해보세요.  
→ `claude mcp list`로 `arha-runtime`이 등록됐는지 확인하세요.

### Q. 빌드 오류 (TypeScript 에러)
→ 명령 프롬프트에서 아래 순서로 실행하세요:
```
npm install
npm run build
```
