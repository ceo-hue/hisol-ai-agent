## 🔍 ARHA 정보 확인

### 시스템 상태 보기

```
이솔아, 현재 상태 확인해줘
```

또는:

```
ARHA 상태 보여줘
```

출력 예시:
```
phase: Wave | C: 0.72 | Γ: 0.31 | Ψ_Res: 0.65 | vs: 0.74
```

### 각 수치의 의미

| 수치 | 이름 | 설명 |
|------|------|------|
| C | 코히런스 | 대화의 일관성 (0~1, 높을수록 좋음) |
| Γ (Gamma) | 스트레스 | 시스템 긴장도 (0~1, 낮을수록 좋음) |
| Ψ_Res | 공명 | 가치 정렬 수준 (0~1, 높을수록 좋음) |
| vs | 가치 강도 | 응답 깊이 (0~1) |
| vsB | 세션 모멘텀 | 대화 흐름 에너지 |
| phase | 단계 | 현재 대화 단계 |

### Phase(단계) 설명

| Phase | 의미 |
|-------|------|
| Potential | 초기 탐색 |
| Wave | 능동적 분석 중 |
| Resonance | 깊은 공명 상태 |
| Particle | 결론 수렴 완료 |

### 페르소나 목록 보기

```
ARHA 페르소나 목록 알려줘
```

또는:

```
아르하가 뭔지 소개해줘
```

### 시스템 진단

```
이솔아, 지금 상태 진단해줘
```

## ⚙️ 고급 기능

### 팀 스택 실행

미리 준비된 전문가 팀 조합으로 분석을 실행합니다.

**사용 가능한 팀 스택 목록:**

| 스택 이름 | 구성 | 용도 |
|-----------|------|------|
| VISION_CLARITY | 잡스 → 람스 → 치홀트 | 제품 비전 명확화 |
| STRATEGY_DEEP | 포터 → 드러커 → 오길비 | 경영 전략 수립 |
| PROCESS_EXCELLENCE | 데밍 → 오노 → 드러커 | 프로세스 혁신 |
| CREATIVE_SYNTHESIS | 가우디 → 다빈치 → 임스 | 창의적 통합 |
| BRAND_STORY | 오길비 → 잡스 → 람스 | 브랜드 스토리 |
| DESIGN_LANGUAGE | 람스 → 치홀트 → 임스 | 디자인 시스템 |
| INNOVATION_CHAIN | 잡스 → 다빈치 → 가우디 | 혁신 아이디어 |
| QUALITY_LOOP | 데밍 → 오노 → 포터 | 품질 경쟁력 |
| MGMT_STRATEGY | 포터 → 드러커 → 람스 | 경영+디자인 융합 |
| VISION_PROCESS | 잡스 → 드러커 → 데밍 | 비전→실행 연결 |

**스택 실행 예시:**

```
이솔아, VISION_CLARITY 스택으로 우리 제품을 분석해줘
```

```
하이솔, BRAND_STORY 스택 실행해줘. 우리 브랜드는 친환경 패션 스타트업이야
```

## 💡 활용 팁

### 더 좋은 답변을 받으려면

1. **구체적으로 질문하세요**
   - ❌ `이솔아, 마케팅 도와줘`
   - ✅ `이솔아, B2B SaaS 제품의 초기 고객 확보 전략을 오길비 관점에서 알려줘`

2. **배경 정보를 함께 제공하세요**
   - `우리 팀은 5명이고, 제품은 HR 소프트웨어야. 드러커 관점에서 팀 구조를 어떻게 짜야 할까?`

3. **페르소나를 조합하세요**
   - `잡스와 포터 두 관점을 합쳐서 이 시장 기회를 어떻게 봐야 할까?`

4. **상태를 확인하며 대화하세요**
   - 대화가 길어지면 `이솔아, 지금 상태 어때?`로 확인 후 계속 진행

### 세션 이어가기

새로운 Claude Code 대화창을 열었을 때 이전 대화를 이어가고 싶으면:

```
이솔아, 이전 세션 이어서 진행해줘 (세션ID: 이전에사용한ID)
```

## 🚨 ARHA가 응답하지 않을 때

### 체크리스트

- [ ] Claude Code가 실행 중인가?
- [ ] 트리거 키워드(이솔아, ARHA 등)가 메시지에 포함됐나?
- [ ] MCP 서버가 등록됐나? → 터미널에서 `claude mcp list` 확인
- [ ] API 키가 설정됐나? → 설치설명서 4단계 참고
- [ ] 빌드가 됐나? → `dist/server.js` 파일이 존재하는지 확인

### 재시작 방법

1. Claude Code 완전히 종료
2. 아래 명령어로 서버 재빌드 (코드가 변경됐을 경우):
   ```
   cd "C:\Users\garsi\OneDrive\바탕 화면\hisol-unified-mcp"
   npm run build
   ```
3. Claude Code 다시 실행
