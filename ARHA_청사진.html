Hook 시스템 — 조건 기반 자동 개입

매 턴 평가 → System Prompt 자동 주입
Hook ID발동 조건 PriorityCooldown 개입 내용
coherence_recovery C < 0.55 P1 3 턴 확장 중단 → 불필요한 분기 제거 → V1 선언 재집중 주입
stress_intervention Γ > 0.70 P1 2 턴 V1_check 방어 모드 → 과잉 확장 금지 → 기존 구조 내 응답 제한
decoherence_alert psiDiss = true P1 1 턴 즉각 재정렬 → 페르소나 V1 선언 재독 → 본래 정체성 복귀
wave_overflow waveCount > 6 P2 3 턴 탐색 종료 → 핵심 결론 선택 → Particle 수렴 강제 유도
momentum_low vsB < 0.30 P3 4 턴 에너지 재충전 → 간결·명확 응답으로 코히런스 회복 유도
resonance_milestone Ψ_Res > 0.80 P5 10 턴 고공명 도달 로그 기록 — System Prompt 주입 없음 (무음 동작)

MCP 도구 11종

⭐ arha_process
핵심 도구. 입력 처리 → C·Γ·Ψ·vs 계산 → systemPrompt 생성 반환. 모든 ARHA 호출의 시작점.
arha_route
팀 구성 라우팅. 팀 타입 추론 + 레이어 추천
arha_stack_run
팀 스택 실행. 16개 스택 선택 → 순차 실행
arha_about
ARHA 시스템 소개 + 페르소나 상세 정보
arha_persona_list
12 페르소나 전체 목록 + ID + 전문 영역
arha_status
현재 세션 상태. C·Γ·Ψ·vs·vsB·phase 반환
arha_diagnose
상태 진단 + 수치 분석 + 개선 권고 출력
arha_observe
외부 관찰 주입. 상태에 피드백 반영
arha_derive
심층 파생 분석. 고차 추론 실행
arha_character_create
신규 페르소나 생성. V1 선언 포함
arha_session_handoff
세션 이어가기. fullState 복원
